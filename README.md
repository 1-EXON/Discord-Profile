## Info
확인할 수 있는 정보
- Token (토큰)
- Name (이름)
- Internal IP (내부 아이피)
- External IP (외부 아이피)
- Tag (태그)
- Id (아이디)
- E-mail (이메일)
- Locale (나라 위치)
- Verified (인증을 했는지?)
- Phone (폰)
- Avatar (아바타)
- Flags / Public Flags
- Mfa Enbabled
- NSFW Allowed
